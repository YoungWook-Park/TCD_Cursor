# PRD: 필름 합착 시뮬레이터

포트폴리오 목적 — 제조 설비(HMI · 시퀀스 · I/O · 이력 · 분석)를 end-to-end로 구현한 시뮬레이터.

---

## 용어

| 용어 | 의미 |
|------|------|
| CGO | 상부 자재(필름) |
| OCA | 하부 자재(필름) |
| 합착 | 상·하 자재를 챔버에서 압력·진공 조건 하에 접합하는 공정 (시뮬 ~2초) |
| UVW | 하부 정밀 정렬 3축 |
| ESC Chuck | 정전기 척 — 전압으로 자재 고정 |
| Ready | 로봇·챔버 안전 대기 위치 |

---

## 설비 구성

- **좌측**: 상부 챔버 (ZUpper축), 하부 챔버 (ZLower + UVW 3축)
- **우측**: 로봇 (TCP 통신), Stage1 (상부 필름), Stage2 (하부 필름 / 완제품)
- **셀 크기**: 2000 × 2000 × 2500 mm (티칭 좌표 기준)

---

## 구현 범위

**포함:**
- WPF HMI (자동/반자동/수동 시퀀스, 레시피, 매뉴얼 축 제어)
- 인프로세스 설비 시뮬레이터 (자재 추적, 축 상태, PLC 핸드셰이크)
- ACS SPiiPlus 모션 제어 연동 (실기 또는 시뮬레이터 전환)
- 로봇 TCP 시뮬레이터 서버
- 비동기 CSV 로깅

**미포함 (향후 확장):**
- 실 비전 알고리즘 — UVW 보정은 가상값으로 시뮬
- PLC TCP 소켓 교환 (Bit/Word 맵 전체)
- MongoDB 공정 이력 저장
- Python + React 분석 대시보드
