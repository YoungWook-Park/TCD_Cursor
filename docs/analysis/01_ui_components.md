## 네비게이션 구조

단일 `MainWindow`의 xaml `ContentPresenter`에 뷰모델을 교체하는 방식(MVVM)으로 페이지 전환을 구현한다.

```
MainWindow
  └── ContentPresenter (CurrentContent 바인딩)
        ├── Main      — 장비 개략도 + 자동 사이클 제어
        ├── Recipe    — 레시피 편집·저장
        ├── Manual    — 수동 축 제어
        ├── Robot     — 로봇 상태 모니터링
        ├── Device    — 하드웨어 연결 관리
        └── Settings  — 앱 환경설정
```
