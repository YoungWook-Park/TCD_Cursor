TCD 솔루션 (WPF, 시퀀스/시뮬레이터, LogWriter, Manual 시퀀스, Motor UI 연동)

## 첫 푸시 (GitHub 연동)

1. [GitHub](https://github.com/new)에서 **New repository** 생성
   - Repository name: `TCD_Cursor`
   - Public 선택
   - **Add a README** 체크 해제 (로컬에 이미 코드 있음)

2. 아래 명령에서 `YOUR_GITHUB_USERNAME`을 본인 GitHub 아이디로 바꾼 뒤 실행:

```bash
cd d:\TCD_Cursor
git remote set-url origin https://github.com/YOUR_GITHUB_USERNAME/TCD_Cursor.git
git push -u origin main
```

3. 로그인/토큰 요구 시 브라우저에서 인증하거나 Personal Access Token 사용
